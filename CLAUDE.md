# BadBuddhas Bubbles

## Проект
Telegram Mini App + веб для breathwork практик. Форк от BadBuddhas Breathwork с новым дизайном.

## Стек
Next.js 14, TypeScript, Tailwind CSS, Supabase, Vercel, Telegram Bot API

## Supabase

## Правила
1. git stash before changes, verify clean working tree
2. run npm run build, fix ALL errors before commit
3. Never mix unrelated changes
4. Always commit and push to main
5. Reference PROJECT_RULES.md для архитектуры
